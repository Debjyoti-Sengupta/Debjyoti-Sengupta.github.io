Comic/Anime Character Identification Pipeline using Vision-Language Model for Autolabelling Dataset

Complete ML Based comic/anime character identification pipeline for creating custom dataset and training standard Computer Vision models for identification.

Description: Created a custom dataset by scraping the internet for specific characters and used a Vision-Language Model (CLIP- Contrastive Language-Image Pretraining by OpenAI) to auto-label the dataset. Reduced the manual effort of creating a dataset by ~90%. Images were downloaded and categorically put in respective class folders automatically, following the ImageFolder dataset structure. This dataset was used for training ResNet18 to create an identification Neural Network model. To illustrate the efficiency of automating much of the dataset creation process 16 classes were chosen, each having a max of 200 images. So a relatively small dataset was created using CLIP for autolabelling and yet the final trained NN achieved an accuracy of ~80%.

Code used can be modified to create ANY custom image dataset for fast deployment in scenarios where character/object classification is required.
